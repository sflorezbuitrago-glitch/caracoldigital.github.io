Bienvenido a Reserva de Espacios

Inicia sesión con tu cuenta corporativa @caracoltv.com.co para reservar salas y espacios de trabajo.

Configuración inicial

Pega la URL de tu Google Apps Script desplegado. Solo se hace una vez.

  • Abre tu Google Sheet → Extensiones → Apps Script → pega el código del archivo google-apps-script.js → guarda.

  • Haz clic en Implementar → Nueva implementación → Aplicación web → Ejecutar como: Yo → Quién tiene acceso: Cualquier persona → Implementar.

  • Copia la URL de la aplicación web que aparece y pégala abajo.
